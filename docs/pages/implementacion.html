Instalación y configuración de un servidor FTP

Para instalar y configurar un servidor FTP, es necesario elegir el software adecuado según el sistema operativo y las necesidades del usuario.

Ejemplo: Instalación de FileZilla Server

Requisitos previos:

  • Sistema operativo Windows (7, 10, 11 o Server).
  • Acceso de administrador.

Pasos de instalación:

  1. Ir al sitio oficial: https://filezilla-project.org/ y descargar FileZilla Server.
  2. Ejecutar el instalador y seguir las instrucciones.

Configuración del servidor:

  • Iniciar FileZilla Server y definir el puerto de escucha (por defecto 21).
  • Crear usuarios y asignarles permisos en la pestaña “Users”.

Abrir puertos en el firewall:

  • Permitir conexiones en el puerto 21 (FTP) y puertos pasivos en el firewall de Windows.
  • Comando en PowerShell para abrir el puerto:
    netsh advfirewall firewall add rule name="FTP" dir=in action=allow protocol=TCP localport=21

Probar la conexión FTP:

Usar FileZilla Client u otro cliente FTP para probar la conexión ingresando:

  • Servidor: localhost
  • Usuario y contraseña: los definidos en la configuración.

Uso de Dreamweaver para conectarse a un servidor FTP y subir archivos

Adobe Dreamweaver facilita la administración de sitios web permitiendo conectarse a un servidor FTP y subir archivos directamente desde su entorno.

Esto es útil para actualizar sitios web en servidores de hosting sin necesidad de usar clientes FTP externos como FileZilla.
